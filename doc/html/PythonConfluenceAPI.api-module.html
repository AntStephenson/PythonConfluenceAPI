all_of(api_call, *args, **kwargs)

 

Generator that iterates over all results of an API call that requires limit/start pagination.

If the `limit` keyword argument is set, it is used to stop the generator after the given number of result items.

>>> for i, v in enumerate(all_of(api.get_content)):
>>>     v = bunchify(v)
>>>     print('     '.join((str(i), v.type, v.id, v.status, v.title)))

:param api_call: Confluence API call (method). :param args: Positional arguments of the call. :param kwargs: Keyword arguments of the call.
